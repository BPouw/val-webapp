Over deze applicatie

Deze applicatie is gemaakt in het kader van het vak 'Clientside web frameworks' van Avans Hogeschool Breda.

Valorant esports

Dit project is een Valorant esports applicatie, Valorant is een video game die competitief gespeeld kan worden. Het doel van deze website is om deze competitie bij te kunnen houden.
De competitie bestaat uit matches die gespeeld worden, het team dat de meeste rondes heeft wint de match.
Een match bestaat uit 2 teams en ieder team bestaat uit 5 spelers. Iedere match wordt op een map gespeeld.
